# Spelly

A Duolingo-style spelling trainer for UK Year 6 SATS. Open `index.html` in any modern browser to start practising.

## Features

- **170+ words** grouped by spelling pattern (prefixes, suffixes, silent letters, etc.)
- **Multiple modes**: timed practice, exam simulation, mistake review, mixed random
- **Audio**: words spoken aloud using the Web Speech API (best with a UK English voice)
- **Progress tracking**: XP, streaks, mastery levels, and score trend charts (saved in localStorage)
- **Letter tiles, multiple choice, and typed input** question types

## How to use

Open `index.html` directly in a browser, or visit the hosted version via GitHub Pages.

No build step, no dependencies -- it's a single self-contained HTML file.
